# Machida Jazz Archive Principles

Version 0.1

## Mission

Preserve jazz culture as faithfully as possible for future generations.

## Core Principles

### 1. Preserve Original Records

Whenever possible, archive records reproduce the original published information.

No normalization should overwrite original data.

### 2. Separate Facts from Interpretation

Original records are permanent.

Normalization, entity resolution, and AI analysis are stored separately.

### 3. Every Record Has Evidence

Every archive record must reference one or more Source records.

### 4. Permanent IDs Never Change

Names may change.

IDs never change.

### 5. Open Collaboration

The archive is designed for continuous improvement through public contributions and review.

### 6. Human + AI

AI assists discovery and analysis.

Human review approves permanent archive records.
